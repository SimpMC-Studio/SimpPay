SimpPay là plugin Minecraft thanh toán tự động được thiết kế cho các server Việt Nam. Plugin hỗ trợ đa dạng cổng thanh toán bao gồm thẻ cào và QR banking, giúp người chơi nạp tiền dễ dàng và tự động nhận phần thưởng.

## Tại sao chọn SimpPay?

## Yêu cầu hệ thống

- **Server**: Paper 1.21 trở lên (và Folia)
- **Java**: Java 8 trở lên (khuyến nghị Java 21 cho webhook performance)
- **Database**: H2 (embedded) hoặc MySQL/MariaDB
- **Plugins tùy chọn**:
  - PlaceholderAPI (cho placeholders)
  - PlayerPoints hoặc CoinsEngine (cho hệ thống xu)
  - Floodgate (cho người chơi Bedrock)
